## Installation
- Clone Repository
- Go to frontend/mycargonaut and run npm i
- Go to backend and run npm i
### Frontend
#### Start
- ng serve
#### Test
- ng test
#### Lint
- ng lint / npm run lint
### Backend
#### Start
- npm run dev
#### Test
- npm run test
#### Lint
- npx eslint .
### Start Database
- docker compose build
- docker compose up
